IPMath is a php library to get informations about ip addresses, networks and to create subnets.

It was a fun project to understand the entire IP Addresses thing.

The library is incomplete.

## Status

IPv4 part is finished, IPv6 part is not finished.

Feel free to make a pull request. :-)
